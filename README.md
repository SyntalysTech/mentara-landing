# Mentara Landing Page

Landing page for [Mentara](https://mentarapp.com) - Your Offline Second Brain.

## Tech Stack

- Next.js 14
- TypeScript
- React 18

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy on Vercel

This project is configured for deployment on Vercel. Simply connect your GitHub repository to Vercel and it will automatically deploy.

## Pages

- `/` - Landing page
- `/privacy` - Privacy Policy
- `/support` - Support & Contact form
